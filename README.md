# Student Performance Analysis

This project analyzes student performance data using Python and Jupyter Notebook.  
It includes data cleaning, visualization, and performance category analysis.

## Features
- Student marks analysis
- Performance categorization
  - Topper
  - Medium
  - Average
- Data visualization using charts
- Simple statistical insights

## Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## File
- `student_performance.ipynb` – Main notebook containing the analysis code.

## How to Run
1. Install Python and Jupyter Notebook.
2. Install required libraries:
   ```bash
   pip install pandas matplotlibx
3. Open the notebook:
jupyter notebook
4. Run all cells in the notebook.

## Output
  Student category analysis
  Graphical representation of marks
  Performance insights
